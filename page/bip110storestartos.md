How to use the community market place to install BIP 110 on [startos](https://start9.com) (0.3.5.1)
===

1. Login into your startos server.
2. Go to "system" and click 5 times on "manage" (You should see "dev tools unlocked" banner)
3. Go to "marketplace" -> "change" -> "add custom registry"
4. Click on Bitcoin -> "Bitcoin Knots (BIP-110 UASF)" -> "reinstall"

You are now running BIP 110! 🎉
